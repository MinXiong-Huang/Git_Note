Git指令紀錄:

複製專案
git clone ~

配置帳戶
git config --global user.name ""
git config --global user.email ""

確認帳戶
git config --list

查看目前狀態
git status

加入檔案
git add .

儲存檔案
git commit -m ''

比較差異
git diff

捨棄所有未commit的修改
git reset --hard

查看歷史
git log

重返某個commit時的狀態
git checkout "hash"

上傳檔案
git push

更新檔案
git pull
